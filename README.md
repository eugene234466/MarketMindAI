
# MarketMind

MarketMind is an AI-powered platform that helps entrepreneurs validate business ideas using real market insights. It analyzes a startup idea to estimate market potential, competition level, niche opportunities, and projected sales performance.

Built with **Python and Flask**, and powered by **Google Gemini**, MarketMind helps founders make **data-driven decisions before investing time or money into a business idea**.

## Features

* AI-powered business idea analysis
* Market research insights
* Competitor analysis
* Sales forecasting
* Niche opportunity detection
* Dashboard with structured results
* Downloadable research reports

## Tech Stack

* **Backend:** Python, Flask
* **Frontend:** HTML (Django-style templates)
* **AI Engine:** Google Gemini
* **Database:** SQLite

## Installation

```bash
git clone https://github.com/eugene234466/marketmindai.git
cd marketmind
pip install -r requirements.txt
python app.py
```

Open in browser:

```
http://localhost:5000
```
